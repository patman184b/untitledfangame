# Raw Inputs

Place source datasets in this directory:

- `natural_earth_terrain.geojson`
- `orbis_settlements.csv`
- `pleiades_sites.csv`

## CSV formats

`orbis_settlements.csv`:

```csv
name,lat,lon
Roma,41.9,12.5
Byzantion,41.0,28.9
```

`pleiades_sites.csv`:

```csv
title,lat,lon
Alexandria,31.2,29.9
Antioch,36.2,36.1
```
